## üéØ How to Play

1. Click "Start Game" to begin
2. Drag your mouse across apples to select them
3. The selection box turns **red** when the sum equals 10
4. Release to score points and remove the apples
5. Maximize your score before time runs out!

## üöÄ Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yhw032/apple-game.git
   ```

2. Navigate to the project directory:
   ```bash
   cd apple-game
   ```

3. Open `index.html` in your browser:
   - Double-click the file, or
   - Use a local server (recommended):
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js
     npx serve
     ```

4. Visit `http://localhost:8000` in your browser
